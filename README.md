# Pandora Boutique — Sitio web

Sitio web tipo catálogo (responsive, con carrito y checkout por WhatsApp) para Pandora Boutique.

## Contenido

- `index.html` — sitio completo autosuficiente (HTML + CSS + JS + logo, todo en un solo archivo). Ábrelo con doble clic para verlo funcionando, sin instalar nada.

## Datos de ejemplo a revisar/actualizar

- Número de WhatsApp para pedidos: +51 981 004 005
- Instagram: @pandoraboutique
- Las imágenes de producto son placeholders (iniciales sobre color) — reemplazar por fotos reales cuando estén listas.

## Publicarlo en internet

Este sitio es 100% estático (no necesita servidor ni base de datos), así que se puede publicar gratis en varios servicios, por ejemplo:

- **GitHub Pages**: sube este repositorio a GitHub y activa Pages en Settings → Pages, eligiendo la rama `main`.
- **Netlify** o **Vercel**: conecta el repositorio de GitHub y despliega automáticamente.
